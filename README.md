# hahatonMIPT
Rusal foil case for MIPT h4k4t0n.  Aerocondensate Team

0) скачать templates, app.py, opt.py (и опционально xlsx для проверки)
1) Запуск локального сервера - через app.py
2) Зайти на http://127.0.0.1:5000/
3) Загрузить исходный xslx файл и запустить расчёт

P.S. Файлы logic, logic2, logic3 уже не нужны для расчёта

Для проверяющих из Rusal: уже обновили версию, скрины с загруженного pdf не очень актуальны
